설치 불필요 · 비밀번호 입력 없음 · 내 브라우저에서만 동작

어떻게 동작하나요?

아래 버튼을 북마크바에 한 번만 등록해두면, Threads에서 그 북마크를 누를 때마다 화면에 작은 패널이 나타납니다. 패널에서 스캔 → 목록 확인 → 언팔을 클릭만으로 진행합니다. 모든 동작은 내 브라우저 안에서만 이루어지며, 어떤 정보도 외부로 전송되지 않습니다.

1단계 · 버튼을 북마크바에 등록

🧹 뒷삭 제거기

👆 이 버튼을 북마크바로 드래그하세요
(북마크바가 안 보이면 Ctrl+Shift+B)

드래그가 안 되나요? (수동 등록 방법)
  1. 아래 [코드 복사] 버튼을 누릅니다.
  2. 북마크바 빈 곳에서 우클릭 → 페이지 추가(북마크 추가)를 선택합니다.
  3. 이름은 "뒷삭 제거기", URL 칸에 복사한 코드를 붙여넣고 저장합니다.

2단계 · Threads에서 실행

  1. 1threads.com에 로그인하고 내 프로필로 이동합니다.
  2. 2북마크바의 🧹 뒷삭 제거기를 클릭합니다.
  3. 3패널에서 [① 스캔 시작]을 누르면 팔로워/팔로잉 목록을 자동 수집합니다.
  4. 4뒷삭 계정 목록이 나오면, 유지하고 싶은 계정은 체크 해제합니다.
  5. 5[② 선택한 계정 언팔]을 누르면 자동으로 팔로우 취소가 진행됩니다.

⚠️ 주의사항

한 번에 너무 많이 언팔하지 마세요. 짧은 시간에 대량 언팔하면 Threads가 계정을 일시 제한할 수 있습니다. 기본 설정(1회 50명, 4~8초 간격)을 권장하며, 대상이 많으면 며칠에 나눠 실행하세요.

  • PC 브라우저(Chrome, Edge, Firefox, Whale)에서 사용하세요. 모바일 앱에서는 동작하지 않습니다.
  • 실행 중에는 다른 탭으로 이동하거나 목록 창을 닫지 마세요.
  • Threads 업데이트로 화면 구조가 바뀌면 동작하지 않을 수 있습니다.
  • 본 도구 사용에 따른 책임은 사용자 본인에게 있습니다.
